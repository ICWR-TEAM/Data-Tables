# Data-Tables
Data tables support for ajax with array javascript format.

Example :

```javascript
json_data = [
    [
        "1",
        "Tes data table", 
        'This Data Table', 
        [
            ['asdas', 'https://incrustwerush.org', 'btn btn-primary mx-1'],
            ['asdas', 'https://www.google.com', 'btn btn-danger mx-1']
        ],
        "tes"

    ]
];

create_table('tables', json_data);
```
